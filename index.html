Sobre os Algoritmos



O dataset processado foi submetido a algoritmos de árvore de decisão.
Foram utilizados os algoritmos J48 do Weka e o Simple CART do Weka.
O dataset foi processado de diferentes formas para a verificação das
melhores condições para a criação da melhor árvore de decisão.

Weka J48

O algoritmo J48, desenvolvido por Ross Quinlan, constrói árvores de decisão a partir de um conjunto de dados de treinamento. As árvores de decisão geradas pelo algoritmo podem ser utilizadas para classificação, logo são conhecidas como classificadores estatísticos. A direita é mostrado o output do algoritmo quando este é rodado sobre todos os dados, com os parâmetros tal, tal e tal. A esquerda estão sendo usados os mesmos parâmetros, só que alguma coisa foi alterada.
